## 示例列表

### `hello_cherryquant.py`
**难度**: ⭐ 入门

**描述**: 最简单的 CherryQuant 程序，演示如何初始化项目并输出基本信息。

**学习要点**:
- 项目导入方式
- 基本日志配置
- 环境变量加载

**运行方式**:
```bash
uv run python examples/01_basics/hello_cherryquant.py
```

**预期输出**:
```
🍒 欢迎使用 CherryQuant AI 量化交易教学项目！
✅ 环境配置加载成功
📊 支持的期货品种: rb, hc, i, j, ...
```

## 常见问题

**Q: 运行示例时提示 "Module not found"?**

A: 确保你使用 `uv run` 命令运行，这会自动设置 PYTHONPATH。

**Q: 如何修改日志级别?**

A: 编辑 `.env` 文件，设置 `LOG_LEVEL=DEBUG`。

**Q: 示例需要配置真实的 API 密钥吗?**

A: 本目录下的基础示例**不需要**真实 API 密钥，使用默认配置即可运行。
