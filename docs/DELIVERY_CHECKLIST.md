# CherryQuant 客户交付清单

> 在将系统交付给客户前，请确保完成以下所有检查项

---

## 📋 交付前准备

### 1. 文档准备
- [ ] 已提供[快速入门指南](QUICK_START.md)
- [ ] 已提供[完整用户指南](USER_GUIDE.md)
- [ ] 已提供[数据下载指南](DATA_DOWNLOAD_GUIDE.md)
- [ ] 已准备技术支持联系方式

### 2. 系统安装
- [ ] 已在客户环境安装Python 3.12+
- [ ] 已安装uv包管理器
- [ ] 已安装Docker和docker-compose
- [ ] 已克隆CherryQuant代码到客户服务器/电脑
- [ ] 已运行 `uv sync --group dev`
- [ ] 已运行 `uv run pip install -e .`
- [ ] 已启动数据库服务（docker-compose up -d）
- [ ] 数据库初始化成功（market_data表已创建）

### 3. 账户配置
- [ ] 客户已注册OpenAI账户并获得API密钥
- [ ] 客户已注册Tushare Pro并升级到2000+积分
- [ ] 客户已注册SimNow模拟账户（用于测试）
- [ ] 已协助客户完成.env配置文件
- [ ] 已验证所有API密钥可用

### 4. 数据准备
- [ ] 已下载历史数据（至少日线数据）
- [ ] 已验证数据库中有数据（SELECT COUNT(*) FROM market_data）
- [ ] 数据格式符合VNPy标准（特别是CZCE 3位数字格式）

### 5. 策略配置
- [ ] 已根据客户需求配置 `config/strategies.json`
- [ ] 品种池设置合理（建议2-3个品种）
- [ ] 风险参数设置保守（单笔风险1-2%）
- [ ] 初始资金设置合理

### 6. 系统测试
- [ ] 已成功运行 `uv run run_cherryquant_complete.py`
- [ ] 系统能正常启动（无报错）
- [ ] Web监控界面可访问（http://localhost:8000）
- [ ] Grafana可访问（http://localhost:3000）
- [ ] 数据库连接正常
- [ ] AI决策日志正常生成
- [ ] 风险管理系统运行正常
